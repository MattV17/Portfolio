# Portfolio
## Portfolio Website: Kurt Matthew V. Villena

This repository contains the source code for my personal portfolio website, designed to showcase my academic background in Computer Science, my technical expertise in Machine Learning, and my professional projects.

---

### Core Objectives

The primary goal of this website is to serve as a centralized hub for my professional identity, providing:

* **Project Documentation:** Detailed insights into my work with AI and data science, including my **Smart Waste Sorter** and **Prediquake** projects.


* **Technical Proficiency:** A visual representation of my skills in **Python, Java, SQL**, and **Machine Learning** frameworks.


* **Academic Milestones:** Information regarding my ongoing Bachelor of Science in Computer Science at **National University** and my thesis on **Automatic Speech Recognition**.



---

### Website Features

* **Interactive Resume:** A digital version of my CV, highlighting my education, creative experience as a **Creative Officer**, and technical certifications.


* **Machine Learning Showcase:** Dedicated sections for ML projects featuring real-time object detection using **YOLOv11** and seismic classification.


* **Responsive Design:** Optimized for a seamless viewing experience across desktops, tablets, and mobile devices using **HTML, CSS, and Figma**.


* **Contact Integration:** Easy-to-use links for professional networking through **LinkedIn** and direct email.
